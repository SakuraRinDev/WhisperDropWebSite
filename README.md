# WhisperDrop — Official Website

The official product page for [WhisperDrop](https://github.com/SakuraRinDev/WhisperDrop), a local-first voice-to-text tool.

🌐 **https://whisperdrop-web-site.vercel.app/**

## About

WhisperDrop turns your voice into text — entirely on your device. Press a shortcut, speak, and the transcription appears at your cursor. No cloud, no uploads, no compromise.

## Tech Stack

- Single-page static HTML (no build step)
- WebGL parallax hero with Parallax Occlusion Mapping
- WebGL click-ripple overlay shader
- i18n (Japanese / English)
- Deployed on Vercel

## Local Development

```bash
python -m http.server 8001 --directory .
```

Then open `http://localhost:8001/`.
